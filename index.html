Software Developer

Hello, I’m Diogo and this is my professional page. I created this page to disseminate my projects and to promote myself as a software developer who's prepared to handle with un-stack development and make ideas come true.

Currently I’m software developer at Philips Informatics where I'm a full-developer at the team RIS&LIS, dealing with the laboratory sector. Before I started work here, I worked for a year at Siscobra Sistemas LTDA, where I had the chance to work with full development projects for the company, developing since the database structure till to the front-end and the back-end. And now I'm also searching about gaming development as a enthusiastic to build a indie game with a team made by me and my friends, Koala Games.

Now let’s talk about my experience in software development and research: I developed a web responsive application to indicate the less dangerous routes to a trip based on federal highways accidents. To do this, I’ve used the Google Maps Javascript API, HTML5, Java EE, Bootstrap, JSF and PostgreSQL as SGBD.

As a academic I've worked at some projects like building software for the third sector; building a quiz game for the university; building a game which aims to simulate de various centers of the university that I have studied, and I also worked developing the SIGMAOn - Geographic Information System for Online Flood Monitoring - which simulate the resulting flood based on river level and the elevation.
